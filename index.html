Général :

* Selecteur universel

h3 em Tous les em dans un h3

h3 + p Le premier p dans h3

a[title] Tous les liens avec title

class ou id (.nomclasse ou #nomid)

INDIQUER LA TAILLE EN EM

font-family Pour selectinner la police, pls peuvent etre mises à la suite

font-style (italic/oblique/normal/bold/)

text-decoration (underline/line-through/overline)

Alignement :

text-align (left/center/right/justify)

Les flottants :

float (left/right)

ex :

html :

<img src="flash.gif" class="imageflottante" alt="Image flottante" /> Ceci est un texte normal de paragraphe, écrit à la suite de l'image et qui l'habillera car l'image est flottante.

css : .imageflottante

{ float: left; }

Pour faire revenir le texte en dessous :

clear both

ex:

html :

<img src="flash.gif" class="imageflottante" alt="Image flottante" />

Ce texte est écrit à côté de l'image flottante.

<p class="dessous">Ce texte est écrit sous l'image flottante.</p>

css :

.imageflottante

{

float: left;

}

.dessous

{

clear: both;

}

body {

background-image: url

}

background-attachment (fixed/scroll)

background-position (top/bottom/left/center/right ou ": 30px 50px;" affichera 30px de la gauche et 50px du haut + possible de combiner ex : top right;)
